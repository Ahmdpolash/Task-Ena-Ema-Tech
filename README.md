## Task Mangement Wesbite : 

## Project setup

```bash
$ git clone https://github.com/Ahmdpolash/Task-Ena-Ema-Tech.git
```
### Server setup -
- open folder in vs code
```
cd server
```
```
npm i
```
- run command
```
npm run start:dev
# for watch
npm run watch
```


### client setup -
- change direcotory
```
cd client
```
```
npm i
```
- then write this command to run this project
```
npm run dev
```
