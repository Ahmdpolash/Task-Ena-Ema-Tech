## Project setup
### Server setup -
```bash
$ git clone https://github.com/Ahmdpolash/Task-Ena-Ema-Tech.git
```
- open in vs code
```
cd server
```
```
npm i
```
- run command
```
npm run start:dev
```


### client setup -
- change direcotory
```
cd client
```
```
npm i
```
- then write this command to run this project
```
npm run dev
```
