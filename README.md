## Project setup

``` bash
$ git clone https://github.com/Ahmdpolash/Task-Ena-Ema-Tech.git
```
## Server setup
- open folder in vs code 
```bash
cd server
```
```
npm i
```

```bash
# development
$ npm run start:dev

# watch mode
$ npm run watch

# production mode
$ npm run start:prod
```

## client setup
